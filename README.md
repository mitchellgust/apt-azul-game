#### Running the application
- If you have a bash-compatible (Linux/Mac/WSL) terminal
```
chmod +x run.sh (only need to do this once)
./run.sh
```
- Else if you have a Windows-based cmd
```
g++ -Wall -Werror -std=c++14 -O -o out/program *.cpp &&
cd out &&
./program <parameters> &&
del /f program &&
cd ..
```
